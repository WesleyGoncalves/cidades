# AppCidades

This is a simple Angular app to manage Continents, Countries and Cities.

## Main Techs

- Angular
- TypeScript
- Angular Material
- SCSS

### Resources

- Responsive design with Angular Material
- Full CRUD for Continents, Countries, and Cities
- Integration with REST API
- Modular architecture
- Reactive programming
- State management with Services and Observables

## Install dependencies

```sh
npm install 
```

## Run in dev mode

```sh
ng serve
```
